# Sisyphus Simulator
## running
* clone repository 
* cd into root folder (outside of /game/)
``` shell     
    $ java game/main.java
```   


## controls
controls are handled through terminal text input, multiple inputs can be entered at a time, inputs are executed in the order they are typed

* wasd: move
* r: refresh map
